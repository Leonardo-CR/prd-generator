# 🚀 AI PRD Generator Skill

Este repositorio contiene una **Skill especializada** diseñada para transformar ideas de software en **Documentos de Requisitos del Producto (PRD)** estructurados, profesionales y optimizados para el desarrollo asistido por IA.

## 🌟 ¿Qué hace esta herramienta?

Utiliza un flujo de trabajo de tres fases para guiarte desde una idea vaga hasta un plano técnico completo:

1.  **Entrevista Inteligente:** La IA te hace preguntas clave sobre tu problema, usuarios, alcance y tecnología.
2.  **Generación de PRD:** Crea un documento Markdown con objetivos, modelos de dominio, requisitos funcionales y criterios de aceptación.
3.  **Optimización para Pipeline:** El documento resultante está diseñado para ser consumido por otras IAs en fases posteriores (definición de MVP, User Stories y Coding).

## 🛠️ Cómo usarla

Si estás usando un asistente de IA compatible con este formato de habilidades (como Antigravity), puedes activar el generador usando frases como:

- *"Quiero hacer una app"*
- *"Genera un PRD para mi proyecto"*
- *"Ayúdame a planear mi software"*
- *"Crea el documento de requisitos de [nombre de tu idea]"*

## 📂 Estructura del Repositorio

- `prd-generator/SKILL.md`: El corazón del proyecto. Contiene las instrucciones, fases y plantillas que sigue la IA.
- `prd-generator/example/`: Contiene ejemplos de PRDs generados con esta herramienta para que veas el resultado final.
- `readme.md`: Esta guía.

## 📝 Ejemplo de Salida

El generador produce documentos con secciones estandarizadas como:
- **Domain Model:** Tablas de entidades y relaciones.
- **Functional Requirements:** Requisitos numerados (FR-01, FR-02...) para trazabilidad.
- **Technical Constraints:** Stack tecnológico y restricciones de despliegue.
- **Acceptance Criteria:** Checklist para verificar que el proyecto está terminado.
